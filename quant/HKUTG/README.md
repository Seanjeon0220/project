# HKU Trading Group

Work done as a member of the HKU Trading Group, a student quant society. Three tracks, in rough order of weight:

- **[`optionpricing/`](optionpricing/)** — the main project, run over two semesters: from American option pricing under Black–Scholes to stochastic local volatility, and then to the question of whether the gap between a calibrated model and the market can actually be traded. It cannot, and working out exactly why is most of the project.
- **[`sectorRotation/`](sectorRotation/)** — treating weekly sector rotation as a Markov chain and testing whether it has any memory to trade on. It does not; most of the work is in trying to overturn that result. Ongoing.
- **[`training/`](training/)** — exercises from the society's curriculum, each implementing one method from scratch: normality testing, kernel regression, Longstaff–Schwartz, ARIMA, uncertain volatility, smile calibration, factor models.
